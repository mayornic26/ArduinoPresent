Arduino Project Lessons by Nico

What is Arduino?

Arduino is an open-source electronics platform based on easy-to-use hardware and software, designed for building digital devices and interactive objects.

How to Open Arduino IDE

  1. Connect your ReadBoard to your computer.
  2. Search for Arduino IDE download
  3. Go to: Arduino Software Download
  4. Install version 2.3.8 for your operating system
  5. Open the IDE after installing
  6. Go to Tools -> Board -> Select Arduino Uno
  7. Select the correct Port after plugging in USB

Wiring Tips:

  • Wire color does not matter, but typically red = positive and black = negative.
  • LED long leg = positive, short leg = negative.

Lesson 1: Blinking LED

Connect GND to the negative rail and 5V to the positive rail. Connect a digital pin to the breadboard. Place the LED with the long leg connected to the signal wire. Add a resistor to the short leg and connect it to ground.

  • Connect GND to the negative rail and 5V to the positive rail.
  • Connect a digital pin to the breadboard.
  • Place the LED with the long leg connected to the signal wire.
  • Add a resistor to the short leg and connect it to ground.

Code:

void setup(){
  pinMode(13, OUTPUT);
}

void loop(){
  digitalWrite(13, HIGH);
  delay(1000);
  digitalWrite(13, LOW);
  delay(1000);
}

Lesson 2: Potentiometer

A potentiometer is a variable resistor that changes resistance.

  • Middle pin -> A0
  • Left pin -> Positive rail
  • Right pin -> Negative rail

Connect LED with resistor to ground and signal pin.

Lesson 3: Servo Motor

A servo is an actuator used to control position.

  • Red wire -> 5V
  • Black wire -> GND
  • Signal wire -> Pin 9 (number of pin does not matter)

Code:

#include 

int potPosition;
int servoPosition;

Servo myservo;

void setup() {
  myservo.attach(9);
}

void loop() {
  potPosition = analogRead(A0);
  servoPosition = map(potPosition, 0, 1023, 20, 160);
  myservo.write(servoPosition);
}

Lesson 4: Button Input

This project teaches how to read input from a button and control an LED.

  • One side of button -> 5V
  • Other side -> Pin 2
  • Resistor (10kΩ) from Pin 2 -> GND
  • LED long leg -> Pin 13
  • LED short leg -> resistor -> GND

Code:

int buttonPin = 2;
int ledPin = 13;
int buttonState = 0;

void setup() {
  pinMode(buttonPin, INPUT);
  pinMode(ledPin, OUTPUT);
}

void loop() {
  buttonState = digitalRead(buttonPin);

  if (buttonState == HIGH) {
    digitalWrite(ledPin, HIGH);
  } else {
    digitalWrite(ledPin, LOW);
  }
}

Lesson 5: Buzzer Sound

This project uses a buzzer to create sound using different tones.

  • Positive leg of buzzer -> Pin 8
  • Negative leg -> GND

Code:

int buzzer = 8;

void setup() {
  pinMode(buzzer, OUTPUT);
}

void loop() {
  tone(buzzer, 1000);
  delay(500);

  noTone(buzzer);
  delay(500);

  tone(buzzer, 1500);
  delay(500);

  noTone(buzzer);
  delay(1000);
}
