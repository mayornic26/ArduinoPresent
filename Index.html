Arduino Project Lessons by Nico

What is Arduino?

Arduino is an open-source electronics platform based on easy-to-use hardware and software, designed for building digital devices and interactive objects.

How to Open Arduino IDE

  1. Connect your ReadBoard to your computer.
  2. Search for Arduino IDE download
  3. Go to: Arduino Software Download
  4. Install version 2.3.8 for your operating system
  5. Open the IDE after installing
  6. Go to Tools -> Board -> Select Arduino Uno
  7. Select the correct Port after plugging in USB

Wiring Tips:

  • Wire color does not matter, but typically red = positive and black = negative.
  • LED long leg = positive, short leg = negative.

Lesson 1: Blinking LED

Connect GND to the negative rail and 5V to the positive rail. Connect a digital pin to the breadboard. Place the LED with the long leg connected to the signal wire. Add a resistor to the short leg and connect it to ground.

Code:

void setup(){
  pinMode(13, OUTPUT);
}

void loop(){
  digitalWrite(13, HIGH);
  delay(1000);
  digitalWrite(13, LOW);
  delay(1000);
}

Lesson 2: Potentiometer

A potentiometer is a variable resistor that changes resistance.

  • Middle pin -> A0
  • Left pin -> Positive rail
  • Right pin -> Negative rail

Connect LED with resistor to ground and signal pin.

Lesson 3: Servo Motor

A servo is an actuator used to control position.

  • Red wire -> 5V
  • Black wire -> GND
  • Signal wire -> Pin 9 (number of pin does not matter)

Code:

#include           //include the servo library

int potPosition;           //this variable will store the position of the potentiometer
int servoPosition;         //the servo will move to this position

Servo myservo;              //create a servo object

void setup() {
  myservo.attach(9);        //tell the servo object that its servo is plugged into pin 9
}

void loop() {
  potPosition = analogRead(A0);                     //use analog read to measure the position of the potentiometer (0-1023)
  servoPosition = map(potPosition, 0, 1023, 20, 160); //convert the potentiometer number to a servo position from 20-160
                                                      //Note: its best to avoid driving the little SIK servos all the
                                                      //way to 0 or 180 degrees it can cause the motor to jitter, which is bad for the servo.
  myservo.write(servoPosition);                      //move the servo to the 10 degree position
}
